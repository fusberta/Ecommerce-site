## Запуск проекта

Чтобы запустить сайт вам нужно:

1.  Скачать проект.
2.  Запустить `index.html` в любом современном браузере, или если у вас установлен пакетный менеджер [npm](https://www.npmjs.com/) прописать в консоли `npm start`.
